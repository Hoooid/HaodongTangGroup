        <div class="np-research-directions">
          <article class="np-research-card">
            <div class="np-research-media">
              <img src="/zh/publication/doi-10-1109-led-2026-3658433/cover.png" alt="高效率倒置 PbS 量子点探测器论文封面">
            </div>
            <div class="np-research-card-copy">
              <span>01</span>
              <h3>胶体量子点短波红外探测</h3>
              <dl>
                <dt>核心问题</dt>
                <dd>如何让溶液法 PbS 量子点同时具备低暗电流、高载流子提取效率和稳定的短波红外响应？</dd>
                <dt>我们的方法</dt>
                <dd>通过配体交换、界面层设计、混合尺寸传输层和能带匹配调控，降低漏电通道并提升载流子传输。</dd>
                <dt>代表成果</dt>
                <dd>近期工作推进了倒置 PbS QD 探测器、界面增强烟雾探测和短波红外传输层优化等方向。</dd>
                <dt>相关论文</dt>
                <dd class="np-paper-links">
                  <a href="https://doi.org/10.1109/led.2026.3658433">LiF/C60 倒置 PbS QD 探测器</a>
                  <a href="https://doi.org/10.1021/acs.jpclett.5c01032">QD 尺寸效应与 SWIR HTL</a>
                  <a href="https://doi.org/10.1109/led.2025.3532323">PVDF 增强 SWIR 探测器</a>
                  <a href="https://doi.org/10.1021/acsphotonics.4c01864">混合尺寸 QD 传输层</a>
                </dd>
              </dl>
            </div>
          </article>

          <article class="np-research-card">

            <div class="np-research-card-copy">
              <span>02</span>
              <h3>钙钛矿与低维光电材料</h3>
              <dl>
                <dt>核心问题</dt>
                <dd>晶体质量、维度、陷阱态和相变过程如何影响新型半导体材料的光电响应？</dd>
                <dt>我们的方法</dt>
                <dd>开展可控生长、缺陷与陷阱态降低、钙钛矿纳米晶向 PbS QD 转化等研究，连接可见光到红外材料体系。</dd>
                <dt>代表成果</dt>
                <dd>钙钛矿单晶薄膜与转化型 PbS QD 为高质量载流子传输和新型探测器提供材料基础。</dd>
                <dt>相关论文</dt>
                <dd class="np-paper-links">
                  <a href="https://doi.org/10.1016/j.xcrp.2023.101363">钙钛矿单晶薄膜</a>
                  <a href="https://doi.org/10.1002/adom.202402740">钙钛矿纳米晶转化 PbS QD</a>
                  <a href="https://doi.org/10.1002/sdtp.17368">CsPbBr3 相变研究</a>
                </dd>
              </dl>
            </div>
          </article>

            <div class="np-research-card-copy">
              <span>03</span>
              <h3>光电探测器阵列与成像</h3>
              <dl>
                <dt>核心问题</dt>
                <dd>如何把单器件性能转化为阵列级均匀性、成像质量和面向应用的传感能力？</dd>
                <dt>我们的方法</dt>
                <dd>发展图案化薄膜、阵列工艺控制、器件到系统测试，以及红外成像和烟雾探测等应用验证。</dd>
                <dt>代表成果</dt>
                <dd>课题组把高性能探测器、微显示和像素集成经验连接起来，形成从器件到成像阵列的研究路径。</dd>
                <dt>相关论文</dt>
                <dd class="np-paper-links">
                  <a href="https://doi.org/10.1021/acsaelm.5c01168">即时烟雾探测应用</a>
                  <a href="https://doi.org/10.1002/jsid.1253">QD-on-silicon microdisplay</a>
                  <a href="https://doi.org/10.1016/j.nanoen.2020.105254">喷涂量子点薄膜</a>
                </dd>
              </dl>
            </div>
          </article>

            <div class="np-research-card-copy">
              <span>04</span>
              <h3>CMOS / 芯片兼容集成</h3>
              <dl>
                <dt>核心问题</dt>
                <dd>哪些材料和界面设计能让胶体光电器件兼容实际芯片和读出电路工艺？</dd>
                <dt>我们的方法</dt>
                <dd>采用自组装单分子层、芯片友好工艺窗口、低温薄膜和可扩展界面堆栈，提升集成可行性。</dd>
                <dt>代表成果</dt>
                <dd>CMOS 兼容 PbS QD 光电探测器将材料化学与集成器件约束直接连接起来。</dd>
                <dt>相关论文</dt>
                <dd class="np-paper-links">
                  <a href="https://doi.org/10.26599/NR.2025.94907796">CMOS-compatible PbS QD PDs</a>
                  <a href="https://doi.org/10.1002/jsid.1253">QD-on-silicon 集成</a>
                  <a href="https://doi.org/10.1109/ogc62429.2024.10738747">电泳沉积 SWIR 探测器</a>
                </dd>
              </dl>
            </div>
          </article>

            <div class="np-research-card-copy">
              <span>05</span>
              <h3>器件物理、表征与系统验证</h3>
              <dl>
                <dt>核心问题</dt>
                <dd>哪些测试真正解释器件瓶颈？不同材料和课题组之间的新型探测器应该如何公平比较？</dd>
                <dt>我们的方法</dt>
                <dd>结合标准化 EQE 思路、瞬态与光谱测试、原位结构表征、界面分析和应用级验证。</dd>
                <dt>代表成果</dt>
                <dd>近期观点文章推动新型探测器测试标准化，早期工作则通过 GISAXS、光谱与输运分析连接结构和性能。</dd>
                <dt>相关论文</dt>
                <dd class="np-paper-links">
                  <a href="https://doi.org/10.1038/s41928-025-01446-7">EQE 测试标准化</a>
                  <a href="https://doi.org/10.1021/acsami.9b00889">两步配体交换</a>
                  <a href="https://doi.org/10.1021/acsami.0c12732">QD-gold 界面形成</a>
                </dd>
              </dl>
